# TripCraft — Claude Code Instructions

## Task Completion

When a task from TODO.md is completed, **always** update TODO.md:

1. **Header metadata** — Update "Last Updated" with the task ID and date
2. **Task Index table** — Strikethrough the completed row with `~~`
3. **Sprint Roadmap** — Change `- [ ]` to `- [x]` and append `COMPLETE ✓`
4. **Verified Complete section** — Add a detailed entry describing what was done, files created/modified, and verification results
